## show-version

Shows the name and version of parent package and exits. See Also: [show-help](http://github.com/azer/show-help)

## Install

```bash
$ npm install show-version
```

## Usage

```js
if(argv.version) require('show-version')();
```

will output:

```
[your-package-name] v[version-number]
```
